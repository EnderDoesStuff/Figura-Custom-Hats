# Hats for Figura
This repo only contains hats for the mod figura
NOTHING ELSE

How to install the hats 
-----------------------

Download Figura: https://www.curseforge.com/minecraft/mc-mods/figura and https://modrinth.com/mod/figura/versions
^^ Mod Links
Then once installed, run the game to install the mods files
Then go to "{YourModpackHere/figura/avatars/" 
and drop the zip into the folder, No need to unzip unless you want to modify the hats
